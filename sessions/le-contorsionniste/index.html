Description

Le grimpeur n'a le droit de poser ses pieds que sur les prises qu'il a utilisé avec ses mains. (Inverse de l'exercice pied-main)

Consignes

  • Voie de toutes couleurs, seul le geste compte.

Objectifs

  • Apprendre à faire de nouveaux mouvements
  • Apprendre à monter les pieds

Variantes
