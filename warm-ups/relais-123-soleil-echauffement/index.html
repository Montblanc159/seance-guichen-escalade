Relais 1-2-3 Soleil! (Échauffement)

2026-03-04

Description

Pas d'assurage, tout le monde participe. Les grimpeurs forment deux équipes et se placent derrière les tapis. Chaque équipe se divisent en deux ou trois groupes. Chaque groupe se place devant une voie en file indienne. Pour plus de clarté, faire en sorte que les voies d'une même équipe soient mitoyennes. Les grimpeurs doivent monter à la première dégaine, redescendre, taper dans la main de la prochaine personne dans sa file et s'assoir derrière le tapis. Au tour du prochain dans la file de faire le 123 soleil. La personne qui encadre tourne le dos aux grimpeurs, compte "1-2-3 Soleil !" et se retourne. Lorsqu'elle compte, les enfants ont le droit de bouger. Lorsqu'elle est retournée, si un enfant bouge, il doit recommencer derrière la ligne des tapis. La première équipe dont tous les joueurs sont assis à gagner. Le mieux, selon le nombre d'enfants, est d'avoir plusieurs personnes qui vérifient que personne ne bouge.

Consignes

  • Interdiction de sauter.
  • Interdiction d'aller plus haut que la première dégaine.
  • Si revanche, intervertir le placement des équipes ou changer les personnes de file pour éviter qu'une file grimpe sur la même voie.
